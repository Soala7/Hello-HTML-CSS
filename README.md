## 🛠 Installation & Setup

### 1️⃣ Install VS Code
- Go to [VS Code website](https://code.visualstudio.com/).
- Download for your operating system.
- Install using the default settings.

---

### 2️⃣ Create Your First HTML & CSS Project
1. Open **VS Code**.
2. Create a new folder for your project (e.g. `hello-html-css`).
3. Create two files:
   - `index.html`
   - `style.css`

---

### 3️⃣ Add HTML Code
Inside `index.html`:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello HTML & CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to HTML & CSS learning.</p>
</body>
</html>
---
